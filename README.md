# MVC1 모델을 활용한 프로야구 구단 웹사이트 
> MVC1 모델을 활용하여 프로야구 구단 웹사이트를 구현했습니다.  
> 참고 사이트인 LG트윈스 웹사이트를 최대한 모방해보고자 했습니다.  

## 담당 구현 기능
1. 로그인/회원가입 등의 회원 관리 기능  
2. 공지사항과 자유 게시판 기능  
3. 프로야구 구단 선수 정보 출력 

- - -  
## 사용 기술
- 개발 환경: Windows10
- 사용 도구: Eclipse, DBeaver, Tomcat
- 사용 기술: Java, MySQL, JDBC, DBCP, UML, JSON, JSP, HTML, JS, CSS
